gridb
=====

grid + adb = gridb


Prerequisites
-------------

* pip
* pip install bottle
* pip install P
* pip install requests

* JAVA_HOME is set
* adb is on your system path


Running it
----------

    python server.py [port]

port is optional, is for what port to start port forwarding devices to, defaults to 8090

the 'server' is started on port 8080
